### Xos-Bot

## Feature
- Auto check-in
- Auto draws

## Requirements
- Node.js 18+
- Npm (node package manager)

## Installation
1. Clone this repository
```bash
git clone https://github.com/CryptoKidzz/Xos-Labs.git cd Xos-Labs
```
2. Install dependencies
bash
npm install
```
3. Create screen 
bash
screen -S name screen
```
4. Fill in your bearer token in `token.txt`

5. Run
bash
node index.js
```
